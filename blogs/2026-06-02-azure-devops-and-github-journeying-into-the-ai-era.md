Microsoft is rolling out agentic capabilities on GitHub while supporting hybrid workflows where teams use GitHub repositories alongside Azure DevOps tools. Enterprise Live Migrations in preview enable organizations to migrate multiple repositories with minimal downtime, and new features like Copilot Code Review and Autofix for CodeQL extend AI capabilities to Azure DevOps repositories. Apple Silicon support for Azure Pipelines is also included in the update.
